# babylon-3dtiles-demo

Minimum implementation for a Babylon 3d tiles renderer by extending the classes from [3DTilesRendererJS](https://github.com/NASA-AMMOS/3DTilesRendererJS). Bounding volume math in based on TilesRenderer implementations.

[Babylon Dingo Gap](http://gkjohnson.githu.io/babylon-3dtiles-demo/example/bundle/index.js)

[Babylon Google Photorealistic Tiles](http://gkjohnson.githu.io/babylon-3dtiles-demo/example/bundle/googleMapsAerial.js)
